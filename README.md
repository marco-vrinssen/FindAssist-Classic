# FindAssist-Classic

Target finding and assist macros for World of Warcraft Classic.

## Features

- **Quick Targeting**: Find and target specific players instantly.\n- **Assist Macros**: Streamlined assist functionality for group play.\n- **PvP Optimized**: Tailored for battlegrounds and arena environments.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
